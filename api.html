Example:

#if defined(OS_WINDOWS)
     /* Some Windows-only code */
#endif

Warning

The following is incorrect usage:

#if OS_WINDOWS
    /* Some Windows-only code */
#endif

This will cause compilation error on non-Windows OS, because this macro won’t be defined. Use IS_OS_WINDOWS instead.
